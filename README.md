# 📊 Mis Prácticas SQL

¡Bienvenido a mi laboratorio de aprendizaje de Bases de Datos! En este repositorio comparto mis avances, scripts, consultas y proyectos prácticos enfocados en la administración de bases de datos relacionales y desarrollo en SQL.

## 🛠️ Tecnologías y Herramientas
*   **Motor de Base de Datos:** Microsoft SQL Server
*   **Lenguaje:** T-SQL (Transact-SQL)
*   **Entorno de Desarrollo:** SQL Server Management Studio (SSMS)

## 📁 Estructura del Repositorio
*   `/Consultas_Basicas`: Ejercicios iniciales (SELECT, WHERE, ORDER BY).
*   `/Funciones_Agregacion`: Prácticas con GROUP BY, HAVING, SUM, COUNT, AVG.
*   `/Joins`: Consultas avanzadas combinando tablas (INNER, LEFT, RIGHT, FULL JOIN).
*   `/Proyectos`: Diseños de bases de datos completas y automatizaciones.
